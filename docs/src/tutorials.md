## Running Tutorials

All tutorials are executable Julia scripts. To run a tutorial:

```julia
import Mycelia
include("tutorials/01_data_acquisition.jl")
```

Or run all tutorials:

```julia
include("tutorials/run_all_tutorials.jl")
```
